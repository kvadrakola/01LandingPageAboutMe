# LandingPageAboutMe

This is the individual project "Landing Page (About me)" for the FrontEnd course. The objective is to create a personal presentation web page using native web technologies.

## Tech Stack
* **HTML5** (Semantic structure)
* **CSS3** (Custom styling without libraries or frameworks like Bootstrap or Tailwind)

## Site Structure
The page features a clean structure and complies with all the requirements of the deliverable (`HEADER`, `MAIN`, and `FOOTER`):

* **HEADER:** Contains the profile avatar, the brand name, and a mobile-adapted menu button.
* **MAIN:** Includes the 5 required sections, with tailored and organized texts:
  1. **Presentation (#hero):** Full name and main focus.
  2. **Birthplace (#grdn):** Information about my hometown (Grodno, Belarus) and nationality.
  3. **Studies (#seo):** Details about my background in Computer Engineering and digital optimization.
  4. **Trajectory & History (#logic):** My professional journey from 2008 up to my arrival in Seville.
  5. **Hobbies (#energy):** My passions and the balance between chess, AI, yoga, and cycling.
  6. **Contact (#contact):** A functional contact form integrated with the Formspree service.
* **FOOTER:** Contains copyright information and direct links to my networks (Telegram and GitHub).

## Main Features
* **AI-Generated Images:** All section background images were generated using Artificial Intelligence with a unified urban graffiti style.
* **Accessibility:** Background images implement `role="img"` and `aria-label` attributes to comply with SEO and web accessibility best practices.
* **Responsive Design:** Optimized structure for mobile viewing using Flexbox and conceptual Auto Layout.
